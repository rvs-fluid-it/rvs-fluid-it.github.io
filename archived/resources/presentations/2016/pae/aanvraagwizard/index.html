Aanvraagwizard Production-ready, ACPAAS-integrated out of the box.

Aanvraagwizard is a Java framework for developing ops-friendly, easy-to-integrate, high-performance, RESTful web services by leveraging Enterprise Integration patterns.

Aanvraagwizard pulls together the stable, mature libraries Camel, Activiti, Guice and Apollo from the Java ecosystem into a simple, light-weight package that lets you focus on getting things done.

Aanvraagwizard has out-of-the-box support for sophisticated configuration, integration application metrics, logging, operational tools, and much more, allowing you and your team to ship a production-quality web service embedded in ACPAAS and integrated with external systems in the shortest time possible.

Rundeck Deploy Slideshow Job

  • De job wordt automatisch getriggerd via Bamboo
  • De job kan 'ad hoc' opgestart worden via Rundeck/NetlifyCMS
  • De job delegeert het genereren van de static site naar de Hugo µService
  • De job gebruikt de 'sftp node executor' om de static site op te laden naar het juiste scherm
  • De job brengt de displays navigator µservice op de hoogte van de nieuw opgeladen static site
